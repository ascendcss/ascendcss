## Definir columnas por porcentajes Las columnas se establecen en porcentajes y para ello debe de difinirlas agregando un numero despues de `col-numero`. `col-5`   = `width: 5%`
`col-75` = `width: 75%` Ahora use la siguiente nomenclatura: - Dispositivos peque침os `col-numero` (Esto soporta todos los tama침os). - Otros `breakpoint:col-numero` | Dispositivo | Clase | | --- | --- | | small | col-numero | | medium | md:col-numero | | large | lg:col-numero | | extra large | xl:col-numero | > Recuerda que los numeros van en multiplos 5 de 0 hasta 100. ```html
1

3
4
``` **Resultado:**

## Espacio entre filas y columnas Los espacios se definen a travez de las siguientes clases de CSS: | Clase | Detalles | |---|---|---| | `gutter-numero` | Crea un margen entre filas y columnas | | `cols-gutter-numero` | Crea un margen solo en las columnas | | `rows-gutter-numero` | Crea un margen solo en las filas | > El numero puede ser entre 0 y 20.
