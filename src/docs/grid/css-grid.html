# Espacio entre filas y columnas Los espacios se definen a travez de las siguientes clases de CSS: | Clase | Detalles | |---|---| | `gap-numero` | Crea un espacio entre filas y columnas. | | `cols-gap-numero` | Crea un espacio solo en las columnas. | | `rows-gap-numero` | Crea un espacio solo en las filas. | > El numero puede ser entre 0 y 20. {% endmarkdown %}
